Welcome to the Surveyor API

Please see the repo for the connected Surveyor Client on GitHub


GET-type calls made from the client include:

  • /api/survey/summary - pulls summary data for a specific survey
  • /api/survey/detail/[survey_id] - pulls detailed data for a specific survey
  • /api/responses/count/[survey_id] - provides the number of responses received for a specific survey
  • /api/responses/all/[survey_id] - pulls all responses for all questions for a specific survey
  • /api/responses/question/[question_id] - pulls all responses for a specific question from a survey
  • /api/responses/other/[survey_id] - pulls all other (or open text) responses for a specific survey


POST-type calls made to the api from the client include:

  • /api/addsurvey - pushes a new survey into the surveys table
  • /api/addsurveyquestions - pushes a new set of survey questions into the questions table based on the survey_id provided in the "addsurvey" post
  • /api/sendresponse - pushes a new response into the responses table
  • /api/sendresponsequestions - pushes a new set of responses for questions into the response_questions table based on the response_id provided in the "sendresponse" post
